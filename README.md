# Diabetes Health Data Analysis using Python

## Project Overview
This project performs exploratory data analysis on a diabetes healthcare dataset using Python.

The analysis focuses on identifying trends, distributions, and relationships between various clinical biomarkers such as BMI, HbA1c, cholesterol, and diabetes classification.

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn

## Analysis Performed
Dataset exploration
Missing value analysis
Statistical summary
Diabetes class distribution
BMI distribution analysis
HbA1c vs BMI relationship
Correlation heatmap visualization

## Key Insights
Higher BMI values were associated with elevated HbA1c levels.
Correlation heatmaps helped identify relationships among biomarkers.
Visualization improved interpretation of healthcare data trends.
